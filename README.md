## About Me

I am Ricardo Lin, an Applied Computing student in the UK with a strong interest in **software engineering, AI-assisted systems, and practical product development**.

My work focuses on building systems that are not only technically sound, but also **usable, transparent, and grounded in real-world problems**. I am especially interested in:

- Python-based application development  
- Data validation and spreadsheet workflows  
- Human-in-the-loop AI systems  
- Software architecture and structured system design  
- Research-driven prototyping  

At the moment, I am developing a project around **AI-assisted spreadsheet records preparation for UK MTD VAT reporting**, combining validation logic, anomaly flagging, review workflows, and traceable outputs.

## Featured Project

### [AI-Assisted VAT Preparation Prototype](https://github.com/ricardo-zihao-lin/ai-assisted-vat-prep-prototype)

A local-first Python prototype for spreadsheet-based VAT record preparation in UK Making Tax Digital contexts.

**Highlights**
- CSV and Excel ingestion
- Deterministic validation checks
- IQR-based anomaly flagging
- Human review workflow
- Traceable CSV export outputs
- Local browser UI with Gradio
